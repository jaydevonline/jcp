##clone this gitrepo 
https://github.com/jaydevonline/jcp 
in apache server repo

##setup both the apps
in angular-app
```
	npm install
```  
then
``` 
	npm start
```
the app will run on localhost:8000/index.html


in react-app
```
	npm run setup
```
then
``` 
	npm start
```

the app will run on localhost:3000

-----------------------------------


Route path from
localhost/jcp/react-app/features

will redirect to angular-app route
